Hex Bot workflow｜六边形 Bot 工作流

一个小型六边形 bot 引导生产线：brief → assets → prompts → render → QA → batch learning。
A small hexagonal bot mascot guides the production line from brief to assets, prompts, rendering, QA, and batch learning.

01 Intake｜产品输入

Capture product truth, audience, offer boundaries, claims, and forbidden assumptions.
记录产品事实、受众、转化边界、允许声明和禁止假设。

02 Assets｜素材资产

Rename and map product images, videos, references, and webpages by visual function.
按视觉功能整理产品图、视频、参考素材和网页。

03 Prompts｜分镜提示词

Write beat-level UGC prompts with real model-readable product inputs and QA gates.
用真实模型输入素材写 1–2 秒级 UGC 分镜提示词，并加入 QA。

Demo gallery slots｜Demo 展示位

Use your own hosted videos or GitHub Release assets for public demos.
可使用自托管视频或 GitHub Release 附件展示公开 demo。
